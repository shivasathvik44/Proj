GATE Exam - 20 MCQs

1. The time complexity of binary search algorithm is:

Correct Answer: O(log n)

2. The Laplace transform of sin(at) is:

Correct Answer: a/(s² + a²)

3. In a relational database, which normal form eliminates transitive dependency?

Correct Answer: 3NF

4. The Fourier transform of δ(t) is:

Correct Answer: 1

5. Which data structure is used in recursive function calls?

Correct Answer: Stack

6. Which modulation technique has the highest bandwidth efficiency?

Correct Answer: QAM

7. In operating systems, thrashing is related to:

Correct Answer: Paging

8. Which sorting algorithm is best for nearly sorted data?

Correct Answer: Insertion Sort

9. In control systems, a system is stable if:

Correct Answer: All poles in LHS

10. Which protocol is used to find the MAC address from IP address?

Correct Answer: ARP

11. Which logic gate is known as the universal gate?

Correct Answer: NAND

12. In compiler design, which phase removes left recursion?

Correct Answer: Syntax Analysis

13. In digital communication, the Nyquist rate is:

Correct Answer: Twice the bandwidth

14. In microprocessors, which interrupt has the highest priority?

Correct Answer: TRAP

15. Which scheduling algorithm may cause starvation?

Correct Answer: Priority Scheduling

16. Which of the following is not a deterministic algorithm?

Correct Answer: Randomized QuickSort

17. In probability, variance is defined as:

Correct Answer: E[X²] − (E[X])²

18. In graph theory, a tree with n vertices has how many edges?

Correct Answer: n−1

19. In electrical machines, the slip of an induction motor is:

Correct Answer: (Ns − N)/Ns

20. Which of the following is not a type of semiconductor material?

Correct Answer: Copper
